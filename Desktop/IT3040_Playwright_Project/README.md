## Singlish to Sinhala Automation Testing

### Tool Used
- Playwright (JavaScript)

### System Under Test
https://www.swifttranslator.com/

### Prerequisites
- Node.js installed on the system

### Setup Instructions
1. Open Command Prompt inside the project folder
2. Run:
   npm install
3. Install Playwright browsers:
   npx playwright install

### Run Tests
npx playwright test

### Description
This project automates functional and UI testing for a Singlish to Sinhala
transliteration system. It validates real-time conversion behavior using
Playwright.
